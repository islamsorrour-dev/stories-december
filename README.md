# مكتبة القصص التربوية

مكتبة تفاعلية تحتوي على 50 قصة تربوية للأطفال بثلاث فئات عمرية مختلفة.

## البنية

- **stories-list.html**: الصفحة الرئيسية التي تعرض جميع القصص (50 قصة)
- **story.html**: صفحة القصة الفردية (ديناميكية)
- **stories-data.js**: ملف يحتوي على بيانات جميع القصص
- **stories-list.js**: JavaScript للصفحة الرئيسية
- **story-script-dynamic.js**: JavaScript للقصة الديناميكية

## الفئات العمرية

1. **3-5 سنوات**: قصص بسيطة وممتعة
2. **6-8 سنوات**: قصص تفاعلية وتعليمية
3. **9-12 سنوات**: قصص متقدمة مع أسئلة مقالية

## كيفية الاستخدام

1. افتح `stories-list.html` في المتصفح
2. اختر قصة من القائمة
3. اختر الفئة العمرية المناسبة
4. استمتع بالقراءة والإجابة على الأسئلة

## إضافة قصص جديدة

لإضافة قصة جديدة، قم بتحديث ملف `stories-data.js` وأضف البيانات في نفس التنسيق.

## الصور

ضع صور القصص في مجلد `images/` بالتنسيق التالي:
- `story-{id}-{age}-{page}.png` للصفحات
- `story-{id}.jpg` للصورة المصغرة في القائمة

## الصوتيات

ضع ملفات الصوت في مجلد `audio/` بالتنسيق التالي:
- `story-{id}-{age}-{page}.mp3` لصوت الصفحات
- `story-{id}-{age}-quiz.mp3` لصوت الأسئلة
